This project involved a series of models and value matrices which were later translated into code to test the initial theory. In this project, I was responsible for assigning and refining the matrix values to implement them in the final result. The rest of the work was done collaboratively with the other project creator, with both of us contributing equally to all parts. This network model was developed using Matlab.
